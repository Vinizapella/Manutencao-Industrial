# 🔩 Manutenção Industrial

> Sistema de gerenciamento de manutenção industrial — controle de ordens, equipamentos e equipes em um só lugar.

---

## 🗂️ Sobre

O **Manutenção Industrial** é uma aplicação desenvolvida para gerenciar e organizar as atividades de manutenção em um ambiente industrial, permitindo o controle de equipamentos, ordens de serviço e histórico de manutenções realizadas.

---

## ✨ Funcionalidades

- 🔧 Cadastro de equipamentos e máquinas
- 📋 Abertura e controle de ordens de serviço
- 👷 Gerenciamento de equipes de manutenção
- 📅 Histórico de manutenções realizadas
- 📊 Relatórios de desempenho e falhas

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Função |
|---|---|
| ☕ Java | Back-end |
| 🐬 MySQL | Banco de dados |
| 📦 Maven | Gerenciador de dependências |

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/Manutencao-Industrial.git

# Acesse a pasta
cd Manutencao-Industrial

# Configure o banco de dados
# Crie o banco e ajuste as credenciais no arquivo de configuração

# Execute o projeto
mvn compile
mvn exec:java
```

---

## 🗃️ Banco de dados

```sql
CREATE DATABASE manutencao_industrial;
```

> Configure a URL, usuário e senha no arquivo de conexão antes de executar.
